# docs/archive — superseded documentation

Per `CC-PRINCIPAL-ENGINEER-MANUAL.md` §8.

## What goes here

Going-forward archive for superseded ADRs, SDDs, ship prompts, and design docs that have been replaced or completed.

## What does NOT go here

Pre-Bundle-28 historical archive (mission docs, ship prompts, audits, state-backups, screenshots from the Audit A1 cleanup pass) lives at `/archive/` at the repo root. See `/archive/README.md` if that index exists.

## Naming

`<original-name>-superseded-by-<replacement>.md` — keep the original content; do NOT edit superseded docs.

Or for completed-and-shipped artifacts: prefix with `shipped-YYYYMMDD-` and move the original here once the work is done.
